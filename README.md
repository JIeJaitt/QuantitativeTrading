## 🔧 功能说明
该工具可以获取以下数据并整理成Markdown报告：
数据类型	说明
股票基本信息	行业、上市时间、市值、股本等
实时行情	最新价、涨跌幅、成交量、市盈率、市净率等
30日行情统计	最高/最低价、均价、涨跌幅、成交量均值等
30日历史行情明细	每日开盘、收盘、涨跌幅等详细数据
财务指标	最近4个报告期的财务数据

## 🚀 使用方法
1. 安装依赖：
```
pip install -r requirements.txt
```
2. 运行程序：
```
# 基本用法（以贵州茅台为例）
python stock_fundamentals.py 600519

# 指定输出文件名
python stock_fundamentals.py 600519 -o 茅台报告.md

# 指定获取天数（默认30天）
python stock_fundamentals.py 000001 -d 60
```

3. 常见股票代码示例：
贵州茅台：600519
平安银行：000001
比亚迪：002594
中国平安：601318


## 📄 输出示例
程序会生成类似 600519_report.md 的Markdown文件，包含完整的基本面分析报告，并同时在控制台输出内容。
